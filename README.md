# Characterify


## Fitur
- 4 tes: **MBTI**, **Big Five (OCEAN)**, **Enneagram**, **4 Temperaments**
- Hasil: **chart** + ringkasan + rekomendasi pengembangan diri
- Sebelum mulai tes: isi **nama** dan pilih **jenis kelamin** 
- Hasil bisa di-**unduh sebagai PDF**
- Menu **Belajar**: 25 artikel offline (Bahasa Indonesia)
- Menu **Web version**: membuka versi web di browser 

## Cara Menjalankan
```bash
pip install -r requirements.txt
python main.py
```

## Catatan
Aplikasi ini untuk edukasi & refleksi diri, bukan diagnosis klinis.
